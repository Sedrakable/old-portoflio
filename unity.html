Plazma Pong

PlazmaPong.apk
Game Description:
Plazma Pong is about the satisfaction of beating a level that you first thought was unbeatable. Remember the falling block patterns, line the ball up, and shoot! Make sure to have the proper strategy to bounce of walls and avoid bombs. You have 12 amazing levels to go through and a final Boss that will make every other level seem easy.


The Process:
After learning the basics of Unity 2D, C#, and Illustrator, I decided to work on a simple project, in terms of gameplay, but that had the functionality that most apps these days have.

Here is what I did step by step:
  1. Worked on physics and gameplay of one level
  2. Added a simple UI
  3. Added sprites to the level and UI
  4. Added more levels and gameplay mechanics
  5. Balanced the levels
  6. Created a level selection page and the sprites needed
  7. Connected the functionality and the progression of the level selection page (storing data)
  8. Added a Boss level, with all the visuals and functionality (took a very long time)
  9. Added an unlimited game mode, Token Grab
  10. Added 4 Power-ups to Token Grab.
  11. Created patterns and a spawner script that would randomly spawn a pattern, but at the same time slowly increasing in speed and difficulty
  12. Added a script that would collect the tokens and store them as currency
  13. Added a Shop where the player can purchase skins and power-up upgrades
  14. Added and Ad mechanic to the Token Grab so that the user can play the game mode a specific amount of times before waiting for a cooldown or watching an Ad
  15. Added a scene where the Player can purchase Token for real-world currency
  16. Cleaned up all the visuals and added new sprites
  17. Added more skins to the store.
  18. Balanced everything
  19. Added an SFX
  20. Added a settings Scene
  21. Checked for Major Bugs

This is a very basic overview of the amount of work that I had to put into this game, there was so much I had to do and learn to make my vision into a reality. I can confidently say that I developed a full-fledged mobile game.

For now the app is not on the Google Play store. Currently I am awating review.
