This is the first game that I created in Java. I had to learn new CSS and Java (JavaFx) skills. I had to really create and position everything from the ground up, from the physics of the cars, the UI, to the pathing, since Java is a very bare-bones programming language, that is not used to make games. One of my rules is to always create my own assets, and that is what I did in this project using Piskel (Pixel Art). Sadly one of the things I did not manage to do at the time, was to create a collision between the cars. Considering this was one of two final projects in college, I still feel proud of what I accomplished.

This is the program that I created as a part of my final project. I had to learn new CSS and Java (JavaFx) skills. I had to base this project on a math or science course, so I chose Linear Algebra. The hardest part of this project was the logic. Making sure that the methods I create are in an order that will properly reduce the Matrix. It was extremely satisfying to see every matrix up to 10 x 10 be able to be reduced.
